# 🎬 Netflix SQL Data Analysis Project

## 📌 Project Overview
This project explores the Netflix dataset using SQL to uncover 
insights about content distribution, popular genres, top 
directors, country-wise production, and more.

---

## 🛠️ Tools Used
- SQL (Microsoft SQL Server)
- Netflix Titles Dataset (Kaggle)

---

## 📊 Business Questions Answered

| # | Question |
|---|----------|
| 1 | Total number of Movies vs TV Shows |
| 2 | Top 5 countries producing most content |
| 3 | All content released in 2020 |
| 4 | Movies directed by Kirsten Johnson |
| 5 | Most common content ratings |
| 6 | TV Shows with 5 or more seasons |
| 7 | Indian movies in the Comedies category |
| 8 | New releases per year (descending order) |
| 9 | Top 5 directors with most movies |
| 10 | Year Netflix added the most content |
| 11 | 5 oldest Indian movies on Netflix |
| 12 | Documentaries released after 2015 |
| 13 | Movie with the longest duration |
| 14 | Most recently released movie per country |
| 15 | Years with 50+ Indian movie releases |

---

## 🔍 Key Insights

- ✅ Netflix has **6,126 Movies** and **2,664 TV Shows**
- ✅ **United States** leads with 3,240 titles
- ✅ **TV-MA** is the most common rating (3,205 titles)
- ✅ **2019** was the peak year for content additions (2,016 titles)
- ✅ **Black Mirror: Bandersnatch** holds the longest duration at 312 min
- ✅ **Rajiv Chilaka** directed the most movies (19 titles)
- ✅ **2017** had the highest Indian movie releases (105 movies)
